## :red_car: 프로젝트 소개

### DALKOM SHOP은 다음카카오(DKT)의 사내 몰
- DalKom.Shop은 DKT 직원들을 위한 전용 쇼핑몰입니다.

- 직원만 가입할 수 있으며, 각 직원은 개인 계정으로 로그인하여 제품 구매가 가능합니다.

- 회사에서 제공하는 마일리지를 활용하여 다양한 상품 및 혜택을 이용할 수 있습니다.
